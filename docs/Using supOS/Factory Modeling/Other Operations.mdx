### Creating Function Set

  
  <ol>
    <li>
      Log in to the supOS platform, and then click{" "}
      <img src="https://freezonex.io/wp-content/uploads/2022/06/7.png" alt="" />{" "}
      to go to the design center.
    </li>
    <li>

      <strong>Function Set </strong>&gt;
      <strong>Function Set</strong>.
    </li>
    <li>
      Select a template, and then click{" "}
      <img
        src="https://freezonex.io/wp-content/uploads/2022/06/26.png"
        alt=""
      />{" "}
      to add a function set.
    </li>
    <li>
      Enter the basic information of the function set and then click{" "}
      <strong>OK</strong>.
    <ul className="li-ul">
    <li>
      Operations of creating a function set are similar to template creation.
      For details, see{" "}
      <a
        href="Creating Object Template#creating-entity-template"
        data-type="internal"
        data-id="#entity_template"
      >
        Creating Entity Template
      </a>
      .
    </li>
    <li>
      A function set can be bound with templates and instances during creation.
    </li>
    <li>
      Select a function set, and then click{" "}

      function set to bound templates and instances.
    </li>
  </ul></li>
  </ol>
 

  
 


:::info
  
When a function set is updated, the supOS platform will not synchronize changes to templates and instances that have bound with it.

### Creating Data Set

  
  <p id="dataSet">
    You can define data sets to bind with events in object models, and receive
    messages related to the bound data set when events are triggered.
  </p>
  
  
  <p className="li-title">Procedure</p>
  
  
  <ol>
    <li>
      Log in to the supOS platform, and then click{" "}
      <img src="https://freezonex.io/wp-content/uploads/2022/06/7.png" alt="" />{" "}
      to go to the design center.
    </li>
    <li>
      Select <strong>Object Model</strong> <strong>Management</strong> &gt;{" "}
      <strong>Data Set</strong>.
    </li>
    <li>
      Click <strong>New</strong>.
    </li>
    <li>
      Enter basic information of the data set, and then click{" "}
      <strong>OK</strong>.
    </li>
    <li>
      Click the data set again, and then click the{" "}
      <strong>Field Definition</strong> tab.
    </li>
    <li>
      Click <strong>New</strong>, enter field property information such as name,
      alias, data type and default value, and then click <strong>OK</strong>.
    </li>
  </ol>

:::info
  
<ul className="note-list">
    <li>Multiple fields are available.</li>
    <li>
      Add fields to bound data sets that have generated data by event
      triggering, data belongs to new fields will be empty.
    </li>
  </ul>

:::
  

  <figure className="wp-block-image size-full">
    <img
      src="https://www.freezonex.io/wp-content/uploads/2022/08/17.png"
      alt=""
      className="wp-image-1945"
    />
  </figure>
  
  
  <ol start={7}>
    <li>
      Click <strong>OK</strong>.
    </li>
  </ol>
 
  
  <p className="li-title">Related Operations</p>
  

  <ul>
    <li>
      Click <strong>Delete</strong> or <strong>View</strong> under{" "}
      <strong>Operate </strong>to delete or view the details of data sets.
    </li>
    <li>
      Click <strong>Edit</strong> or <strong>Delete </strong>under{" "}
      <strong>Operate </strong>on <strong>Field Definition</strong> page to edit
      or delete fields.
    </li>
  </ul>
 


:::info
  
Bound data sets cannot be deleted.

:::

  

  ### Creating Data Group

  
  <p>
    Group and manage data permission of object instance properties. When you
    select different groups, you can get different permissions on instance
    property.
  </p>
  

  <ul>
    <li>
      You cannot see object properties that you have no permission of in design
      center or object selector.
    </li>
    <li>
      You cannot see corresponding data of properties that you have no
      permission of in operation center.
    </li>
  </ul>



:::info
  
Data groups can only be applied to entity instances.

:::

  
  
  <p className="li-title">Procedure</p>
  
  
  <ol>
    <li>
      Log in to the supOS platform, and then click{" "}
      <img src="https://freezonex.io/wp-content/uploads/2022/06/7.png" alt="" />{" "}
      to go to the design center.
    </li>
    <li>
      Select <strong>Object Model</strong> <strong>Management</strong> &gt;{" "}
      <strong>Data Group</strong>.
    </li>
    <li>
      Click <strong>New</strong>.
    </li>
    <li>
      Enter basic information of the data set, and then click{" "}
      <strong>Save</strong>.
    </li>
    <li>
      Select the <strong>Permission Config</strong> tab, click{" "}
      <img
        src="https://freezonex.io/wp-content/uploads/2022/06/26.png"
        alt=""
      />{" "}
      next to <strong>Template Config</strong>to select a template.
    <p className="info">
    Select templates of instances that contain properties you need.
  </p></li>
  </ol>
 
  
  
  
  
  <ol start={6}>
    <li>
      Click the <strong>Base Object</strong> tab, and then select a template to
      configure property permissions.
    <ul className="li-ul">
    <li>
      On the right side, select <strong>All</strong> <strong>Instances </strong>
      to grant permissions to properties of all instances under the template.
    </li>
    <li>
      Click <strong>Partial Instances</strong> to grant permissions to
      properties of selected instances under the template.
    <ol className="li-li-li">
    <li>
      Click <strong>New </strong>to select instances, and then click{" "}
      <strong>OK</strong>.
    </li>
    <li>
      Click <strong>Attributes </strong>under <strong>Operate </strong>to select
      all or partial properties of the instance.
    </li>
    <li>
      Click <strong>OK </strong>after selecting needed properties.
    </li>
    <li>
      Click<strong> Alarm Permission</strong> under <strong>Operate </strong>of
      selected property to configure alarm permissions, and then click{" "}
      <strong>OK</strong>.
    </li>
    <li>
      Click <strong>Close</strong>.
    </li>
  </ol>
</li>
  </ul></li>
  </ol>
 

  
 

  

:::info
  
Alarm data can be viewed in operation center, but not used to build templates.

::: 
    
  

  <figure className="wp-block-image size-large">
    <img
      src="https://www.freezonex.io/wp-content/uploads/2022/08/18-1024x606.png"
      alt=""
      className="wp-image-1946"
    />
  </figure>
  
  
  <p className="li-title">Related Operations</p>
  

  <ul>
    <li>

    </li>
