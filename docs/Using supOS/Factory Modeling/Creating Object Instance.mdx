## Overview

  
  <p>Object instance is a specific instance of an object template.</p>
  
  
  <div className="wp-block-group">
  
    <ul>
      <li>
        Multiple instances can inherit from a same template and link one or more
        function sets.
      </li>
      <li>
        Each instance can have its own property, service, event and subscription
        aside from those inherited from the parent template and function set.
      </li>
    </ul>
 

## Creating Entity Instance
  
    
<p>
      Create an instance of a specific entity such as a collector source point
      or a tag to inherit from the parent template, and then obtain required
      data from custom script or bound data source and service.
    </p>
    

    
<ol>
      <li>
        Log in to the supOS platform, and then click{" "}
        <img
          src="https://freezonex.io/wp-content/uploads/2022/06/7.png"
          alt=""
        />{" "}
        to go to the design center.
      </li>
      <li>
        Select <strong>Object Model</strong> <strong>Management</strong> &gt;{" "}
        <strong>Object Instance</strong>&gt;
        <strong>Entity Template</strong>.
      </li>
      <li>
        Select a template, and then click <strong>New</strong> to add an
        instance.

      <li>
        Enter the basic information of the instance and then click{" "}
        <strong>Save</strong>.

      <li>
        <strong>Alias</strong>: Unique ID of the instance. Tag number is
        generally used for factory equipment.
      </li>
      <li>
        <strong>Code</strong>: Code of the instance. Keep it the same as the
        actual equipment code.
      </li>
    </ul></li>
    </ol>
